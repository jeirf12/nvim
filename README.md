_If it does not exist, create it with the following command:_

```

```
_Then copy the following:_
```
mv ~/vim/conf* ~/.config/nvim/config/
mv ~/vim/init.vim ~/.config/nvim/
```
_Finally copy the **.vimrc** in **.config**:_
```
mv ~/vim/.vimrc ~/.config/
```
