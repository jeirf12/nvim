### Installation ðŸ”§

_First you enter the folder like this:_

```
$ cd vim
```

_Run the following script:_

```
$ bash install.sh
```

_or_

```
$ ./install.sh
```

_(optional)if you want to use the same neovim settings in vim, create a symbolic link file like this:_
```
$ ln -s ~/.config/.vimrc ~/.vimrc
```
